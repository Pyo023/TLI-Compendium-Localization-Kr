This repo contains community translations for the TliCompendium app (Torchlight Infinite VoidChart & Build Planner tool).
Currently supports English (base), Chinese, Taiwanese, Russian, Korean, and Japanese.

Files are saved as .ts files (en.ts, cn.ts, tw.ts, ru.ts, ko.ts, ja.ts).
To contribute: Fork the repo, translate naturally for gamers, then submit a PR.
